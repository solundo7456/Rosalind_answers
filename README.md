# Rosalind_answers

# Rosalind Bioinformatics Projects

This repository contains solutions to **Rosalind bioinformatics challenges**, covering DNA, RNA, protein, and genetics problems.
https://rosalind.info/problems/list-view/

---

## 📂 Project Breakdown

### **Sequence & String Manipulation**
- **Chaining the Amino Acids** – assemble peptides into proteins.  
- **Open Reading Frames (ORFs)** – identify all possible protein-coding sequences.  
- **Consensus Sequences** – compute the most common nucleotide per position.  
- **Overlap Graphs** – construct graphs based on sequence overlaps.  
- **Finding Common Substrings** – detect shared DNA/protein segments.

### **Translation & Transcription**
- **Pitfalls of Reversing Translation** – back-translate proteins into possible mRNA sequences.  
- **Transcription** – convert DNA sequences to RNA.  
- **Protein Motif Search** – locate motifs using shorthand notation.

### **Genetics & Probability**
- **Mendelian Inheritance** – simulate and calculate genotype/phenotype probabilities.  
- **Probability & Expected Value** – solve probability problems in genetics.

### **Sequence Analysis**
- **Max GC Content** – identify sequences with the highest GC proportion.  
- **Nucleotide Count** – count nucleotides in DNA sequences.  
- **Fibonacci Modeling** – simulate population growth and mortality using Fibonacci sequences.
